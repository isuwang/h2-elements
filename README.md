# \<h2-elements\>
A series of container elements to quickly build web apps

快塑网内部系统开发使用的polymer elements。

## 构建方法
### 安装node.js 使用nodeJs默认的npm包管理
### 安装bower
```
npm install bower -g
```
### 安装h2-elements
```
bower install h2-elements --save
```
### 项目地址：https://github.com/isuwang/h2-elements.git
